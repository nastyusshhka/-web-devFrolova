1. Explain “repaint” and “reflow” of the webpage

Answer: Repaint is about updating the appearance, when something changes visually (like color or text), the browser needs to repaint that part of the page. Reflow is when elements on a web page change position or size, the browser needs to recalculate where everything goes. This can happen when you add or remove elements, change styles, or resize the window.

2. How does JavaScript on the webpage affects the page load speed and performance?

Answer: When the browser loads JavaScript files, it can slow down the page loading speed, especially if the JavaScript files are large or a lot, JavaScript is used to add interactivity to the page, which can also slow down the loading process. To improve site loading, you can use loading attribute, minify and compress JavaScript code.

3. What is “minification” of the script? Why it is useful to minify your scripts?

Answer: Minification is the process of minimizing code and markup in your web pages and script files. It's useful, because it helps reduce file size and improve loading.

4. What is a JS Web Worker and when do we need it?

Answer: Web Workers makes it possible to run a script operation in a background thread separate from the main execution thread of a web application.
